# Список моделей

Вывод списка всех доступных моделей из WS в консоль.

Строка кода для получения списка моделей:

```py
models = await ws.get_models()     
for model in models:         
    print(model.info())
```

Где:

* ws - Workspace, к которому ранее подключился пользователь.
